## Beberapa Tipe Operator:

| Operator | Deskripsi                         |
| -------- | --------------------------------- |
| +        | Penjumlahan                       |
| -        | Pengurangan                       |
| \*       | Perkalian                         |
| /        | Pembagian                         |
| -/       | Pembagian mengembalikan nilai int |
| %        | Module atau sisa hasil bagi       |

## Beberapa Tipe Operator Perbandingan:

| Operator | Deskripsi               |
| -------- | ----------------------- |
| ==       | Sama dengan             |
| !=       | Tidak sama dengan       |
| >        | Lebih dari              |
| <        | Kurang dari             |
| >=       | Lebih dari sama dengan  |
| <=       | Kurang dari sama dengan |
